# Smart Job Application Tracker 🚀

## 📌 Project Status

🚧 **In Progress** — Backend development ongoing

---

## 📖 Overview

Smart Job Application Tracker is a backend API that helps users manage and track job applications efficiently.

This project is being built step-by-step as a full-stack application.

---

## ⚙️ Features (Current)

* Basic Express server setup
* API running on localhost
* CORS enabled
* Environment variable support (.env)
* API tested using Thunder Client
* Git & GitHub integration

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* Thunder Client (API Testing)
* Git & GitHub

---

## ▶️ How to Run

1. Install dependencies:

```
npm install
```

2. Start development server:

```
npm run dev
```

3. Open in browser:

```
http://localhost:5000
```
